# Publishing Checklist

Use this when you are ready to publish `opensource-showcase` to npm.

## Repository Visibility

- Keep this CLI source repo private while building.
- Make it public only when you are ready to open-source the tool.
- The generated user showcase repository should be public because it is for GitHub Pages and portfolio sharing.

## Before Publishing

1. Create a GitHub OAuth App:
   https://github.com/settings/developers

2. Copy the app's public Client ID.

3. Replace `Ov23li4P4stPDon9vAt1` in:
   `src/auth/github-auth.ts`

   The Client ID is safe to ship in a CLI. Do not add or ship a client secret.

4. Update package metadata in `package.json`:
   - `author`
   - `repository.url` (already set to https://github.com/opensource-showcase/opensource-showcase.git)
   - `bugs.url` (already set to https://github.com/opensource-showcase/opensource-showcase/issues)
   - `homepage` (already set to https://github.com/opensource-showcase/opensource-showcase#readme)
   - package `name` if `opensource-showcase` is taken

5. Run checks:
   `npm run type-check`
   `npm run build`

6. Test the packed package:
   `npm pack`
   `npm install -g ./opensource-showcase-0.1.0.tgz`
   `opensource-showcase --help`
   `opensource-showcase login`
   `npm uninstall -g opensource-showcase`

## Package Name

Check:
https://www.npmjs.com/package/opensource-showcase

If the name is taken, use one of these:
- `@yourusername/opensource-showcase`
- `opensource-showcase-cli`
- `os-showcase`

For scoped packages, publish with:
`npm publish --access public`

## Publish

1. Login:
   `npm login`

2. Publish:
   `npm publish --access public`

3. Verify:
   https://www.npmjs.com/package/opensource-showcase

4. Test install:
   `npx opensource-showcase`

## Updating Later

1. Make changes.
2. Bump version:
   `npm version patch`
3. Run:
   `npm run type-check`
   `npm run build`
4. Publish:
   `npm publish`

## Troubleshooting

"Package name already exists"
Use a scoped package name such as `@yourusername/opensource-showcase`.

"You must be logged in"
Run `npm login`.

"prepublishOnly failed"
Run `npm run build` locally and fix the TypeScript error.
